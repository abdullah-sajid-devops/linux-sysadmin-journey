## 📖 1. Kernel vs GNU Tools

The **Linux Kernel** is the core engine — it manages hardware, memory, 
and processes. Everything else (shell, utilities, command-line tools) 
comes from the **GNU Project**. What we call "Linux" is technically 
**GNU/Linux** — the kernel plus the GNU tools working together.

traditional **Closed Source** models, where source code stays locked

## 🌳 3. Linux Distribution Lineages

Covered the three major Linux family trees that most modern distros 
trace back to:

- **Debian** → Ubuntu, Linux Mint, Kali Linux
- **Red Hat** → Fedora, CentOS, RHEL
- **Slackware** → SUSE, openSUSE
