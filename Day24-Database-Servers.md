# Day 24 of Cisco Linux Essentials Course

Today I learned another important topic called Database Servers.

## Database Servers
Database servers form the backbone of most online applications. Any dynamic web application pulls data from and writes data to these databases.

**Example:** Think of a web program that keeps records of students — it has a front-end web server that displays a web form on the screen. When you enter data into that form, it gets saved on the backend into a database like MariaDB.

## MariaDB vs MySQL
MariaDB is actually a community-developed fork of MySQL. Just like ownCloud was forked into Nextcloud, the community copied MySQL's open-source code and built MariaDB from it, so that it could remain completely free and open-source.

## Data Retrieval & SQL Language
A database's job isn't just to store information — it also makes it easy to retrieve and search that data whenever needed.

## Interesting Takeaway
MySQL and MariaDB aren't just used in fields like data analytics — they're also heavily used in DevOps, especially when deploying a website or application onto a server.
